# Directives

Standard Operating Procedures (SOPs) for this project.

Each directive is a Markdown file that defines:
- **Goal** — What the directive accomplishes
- **Inputs** — What data/context is needed
- **Tools/Scripts** — Which execution scripts to call
- **Outputs** — What gets produced
- **Edge Cases** — Known pitfalls and how to handle them

Directives are living documents — update them as you learn.
